
# Tools & Tech Stack

## Data Processing
- Python (pandas, numpy) — data cleaning, transformation, validation, deduplication

## Database
- Microsoft SQL Server — centralized storage and reporting layer
- ODBC / pyodbc — connecting Python to SQL Server

## Automation / Scheduling
- Scheduled execution (daily timing)
- Logging + notifications

## Reporting
- Excel dashboards with Power Query (refresh connections + visuals)
- Dashboard exports

## Integrations (private/internal only)
- Browser automation (Selenium) for portal exports
- API calls (requests) for system downloads
